# GitQuest

GitQuest is a small web app to practice Git concepts through visual exercises.
You run Git commands in a terminal and see how the commit graph changes in real time.

The goal of each level is to make the **current repository graph match the target graph**.

## What it helps with

* Understanding commits and history
* Creating and switching branches
* Seeing how merges affect the graph
* Learning how `HEAD` moves

## How it works

1. Read the instructions for the level
2. Run Git commands in the terminal
3. Watch the graph update
4. Match the current graph with the target graph

## Run locally

```bash
npm install
npm run dev
```

Then open the local server in your browser.

## Notes

This project is meant for learning and experimentation.
If something looks wrong in the graph, refresh the level and try again.
